1다이렉트 거울이란?

스마트폰을 손거울처럼 쓸 수 있게 해 주는 앱입니다. 전면 카메라로 실제 거울처럼 내 모습을 비춰 줍니다.

가장 큰 특징은 빠른 실행입니다. 복잡한 잠금 해제나 메뉴 없이, 필요한 순간 한 번에 거울을 켤 수 있습니다. 이 방식은 특허 등록된 독자 기술입니다.

이런 때 편리해요

  • 화장을 고치거나 메이크업을 확인할 때
  • 머리를 정리하거나 외출 전 얼굴을 볼 때
  • 손거울이 없을 때 폰으로 빠르게

2처음 시작하기

앱 실행 — 홈 화면에서 다이렉트 거울 아이콘을 탭합니다.
카메라 권한 허용 — '허용'을 선택합니다. 카메라는 거울 표시에만 쓰이며 사진은 저장되지 않습니다.
거울 시작 — 바로 거울 화면이 나타납니다. 좌우 반전되어 실제 거울처럼 보입니다.
안심하세요이 앱은 카메라 영상을 화면에 보여줄 뿐, 저장하거나 외부로 전송하지 않습니다. 개인정보를 수집하지 않습니다.

3기본 사용법 (화면 설명)

거울 화면 아래에 버튼이 있습니다. 아래 화면의 번호를 참고하세요.

▲ 거울 화면과 버튼 번호
① 홈처음 화면(홈)으로 돌아갑니다.
② 좌우 반전거울 좌우 반전을 켜고 끕니다. (글자 볼 때는 끄기)
③ 그레이 모드카메라 없이 화면 밝기로 은은한 반사면을 만듭니다.
④ 안내빠른 실행 설정 방법을 안내합니다.
⑤ 확대/축소화살표로 줌 조절. 두 손가락으로 벌리거나 오므려도 됩니다.

4빠른 실행 설정 핵심 기능

한 번 설정해 두면 앱을 찾지 않고도 순식간에 거울을 켤 수 있습니다. 거울 화면의 ④ 안내 버튼을 누르면 아래 화면이 나옵니다.

▲ 빠른 실행 설정 안내

측면 버튼 두 번 누르기 (가장 빠름)

설정 열기 — 휴대폰 설정 → 유용한 기능으로 이동합니다.
측면 버튼 선택 — '측면 버튼' 또는 '사이드 키'를 탭합니다.
두 번 누르기 설정 — '두 번 누르기' → '앱 열기' → 다이렉트 거울을 선택합니다.
완료 — 이제 측면 버튼을 두 번 누르면 언제든 거울이 켜집니다.
더 간편하게홈 화면 위젯을 추가하거나, 상단 빠른 설정(QS)에 거울 타일을 추가하면 탭 한 번으로도 실행할 수 있습니다.

5알아두면 편리한 기능

어두운 곳에서 자동 밝기

주변이 어두워지면 화면이 자동으로 밝아져 어두운 곳에서도 얼굴이 또렷하게 보입니다. 따로 조작할 필요 없습니다.

그레이 반사면 모드

카메라 없이 화면 밝기만으로 은은한 반사면을 만드는 모드입니다. 배터리를 아끼며 간단히 거울처럼 쓸 수 있습니다.

화면 꺼짐 방지

거울을 보는 동안 화면이 저절로 꺼지지 않습니다.

6자주 묻는 질문

카메라 영상이 저장되나요?
아니요. 화면에 보여줄 뿐 저장·전송하지 않습니다.
개인정보를 수집하나요?
아니요. 어떤 개인정보도 수집하지 않습니다.
측면 버튼 설정이 안 보여요.
기기·제조사에 따라 메뉴 이름이 다를 수 있습니다. 설정에서 '측면 버튼' 또는 '사이드 키'를 검색해 보세요.
글자가 거꾸로 보여요.
② 좌우 반전 버튼을 눌러 반전을 끄면 똑바로 보입니다.
광고가 있나요?
없습니다. 광고 없이 깔끔하게 사용할 수 있습니다.
